# LGU Student Portal - Frontend Clone

A comprehensive frontend clone of the Lahore Garrison University (LGU) Student Portal created as a semester project.

## Project Description

This project is a frontend-only implementation of a university student portal system, inspired by the LGU student website. It provides a modern, responsive interface for students to access their academic information, courses, grades, and other university services.

## Tech Stack

- **HTML5** - Structure and semantic markup
- **CSS3** - Styling and animations
- **JavaScript** - Interactivity and dynamic functionality
- **Bootstrap 5.3.0** - Responsive layout and components
- **Font Awesome 6.4.0** - Icons

## Features

### Landing Page (index.html)
- Professional hero section with university branding
- Student login form with password toggle
- Feature showcase section
- Latest announcements display
- Quick links to important resources
- Responsive navigation menu
- Social media integration
- Contact information footer

### Dashboard (dashboard.html)
- Comprehensive student dashboard
- Multiple sections:
  - **Overview**: Activity feed, upcoming events, notifications
  - **Profile**: Personal information and academic standing
  - **Courses**: Enrolled courses with progress tracking
  - **Grades**: Semester grades and GPA calculator
  - **Attendance**: Attendance tracking system
  - **Fee Payment**: Fee management portal
  - **Course Registration**: Registration system
  - **Library**: Digital library access
  - **Messages**: Communication system
- Sidebar navigation
- Top navigation with user menu
- Statistics cards with visual indicators
- Interactive course cards
- Grade tracking with GPA display

## How to Use

1. Open `index.html` in a web browser to view the landing page
2. Click the login button (credentials not required - demo mode)
3. Navigate to `dashboard.html` to explore the student dashboard
4. Use the sidebar menu to switch between different sections
5. All interactions are simulated for demonstration purposes
